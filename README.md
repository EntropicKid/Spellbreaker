# Spellbreaker
Spellbook AddOn for World of Warcraft: Aviana RP. Что сейчас имплементировано:

📖 Книга заклинаний с фильтрами по классу, поиск по названию, фильтр «все / кастомные / вшитые».\n
🎯 Подготовка заклинаний через drag-and-drop из библиотеки на главное окно.
🎲 Каст с бросоком d20: GM задаёт DC, модификатор от ранга прибавляется автоматически, поддерживаются критические успехи/провалы.
🛌 Долгий и Короткий отдых: восстанавливают ячейки/рвение, рассылаются по сети (только лидер группы).
👑 Панель Ведущего с очередью заявок, кнопками «Успех / Провал / Крит. успех / Крит. провал» и «+Эффект».
📊 Сетевая синхронизация: класс, ранг, подход, ячейки, рвение, подготовленные заклинания и активные эффекты рассылаются всем участникам группы.
✏️ Конструктор кастомных заклинаний с лимитами символов (UTF-8) и валидацией перед сохранением.
⏳ Активные эффекты (контейнеры): длительные/потоковые ауры с счётчиком применений, поддержкой концентрации, синей рамкой.
📜 Журнал системы с возможностью скрыть системные сообщения в чате игры.
🔘 Кнопка миникарты (LDB) с тултипом и тремя режимами клика.

🧹Защита от спама: лимит 40 кастомных заклинаний на отправителя.
🔧Поддержка .caura toggle для Авианы.

Установка
Из исходников (текущая версия):

git clone https://github.com/<owner>/Spellbreaker.git
cp -r Spellbreaker/Spellbreaker  /path/to/WoW/_retail_/Interface/AddOns/

Из архива:
1.Скачайте последний релиз со страницы Releases.
2.Распакуйте Spellbreaker-vX.Y.Z.zip в <каталог WoW>/Interface/AddOns/.

Управление:
/sb	Открыть/закрыть главное окно

Бинды:
ЛКМ на кнопке миникарты — главное окно.
ПКМ на кнопке миникарты — панель Ведущего (только лидер группы).
Shift + ЛКМ на кнопке миникарты — Библиотека.
ЛКМ по игроку в GM-панели — выдать ресурсы.
ПКМ по игроку в GM-панели — переключить «заклинания ↔ активные эффекты».
Drag карточки заклинания из библиотеки в главное окно — подготовка.
Drag карточки заклинания из главного окна — разучение.
